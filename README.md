## l-vite-cli

### 🧾 功能概要
为了开发项目方便，做的一个Cli。可以在vite项目下，通过命令创建`.vue`文件 创建`router` 创建`pinia`,功能类似于IDE里面代码片段。

### 📦 安装
```
npm install l-vite-cli

yarn add l-vite-cli

bun install l-vite-cli
```

### 🚀 使用
```bash
# 可以在当前目录下的pages下创建 .vue的文件
   l-vite vue index
   
# 可以在当前目录下安装Vue的Router@4
   l-vite router
```
